# Models


This is the models package.

The following models are implemented:
1. [SyConNTM](syconntm.py): SyConNTM model
2. [ETM-D](etm_dirichlet.py): ETM-D model
3. [ETM](etm.py): ETM model
4. [ldamallet](lda.py): LDA model
5. [DVAE](dvae.py): DVAE model
6. [ContextNetwork](lm.py): ContextNetwork model
